┌─────────────────────────────────────────────────────────────────────────────┐
│ EXECUTION STEPS                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

  1. blockMesh
     → Reads system/blockMeshDict
     → Generates ~20,000 hex cells
     → Creates constant/polyMesh/

  2. topoSet
     → Reads system/topoSetDict
     → Creates fan faceSet (searchableDisk)
     → Creates fanZone faceZoneSet

  3. createPatch -overwrite
     → Reads system/createPatchDict
     → Converts fanZone → cyclic patch "fan"
     → Modifies constant/polyMesh/boundary

  4. checkMesh -allTopology
     → Verifies mesh connectivity
     → Checks for duplicates and invalid faces
     → Reports skewness and aspect ratios

  5. foamRun
     → Runs incompressibleFluid solver
     → Transient PIMPLE iterations
     → Writes results every 0.1 s
     → Samples probes at 1, 1.5, 2.5 m downstream

┌─────────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

  PROBLEM: "fan patch not found" error
  ───────────────────────────────────────
  → createPatch -overwrite may have failed
  → Check: grep -n "fan" constant/polyMesh/boundary
  → If missing: manually run "createPatch -overwrite"

  PROBLEM: Solver diverges/NaN values
  ────────────────────────────────────
  → Time step too large or fan pressure jump too high
  → Reduce: deltaT from 0.05 to 0.01 in controlDict
  → Or reduce: f from 4.17 to 2.0 in 0/p
  → Restart and rerun

  PROBLEM: Very slow simulation (>2 hours)
  ────────────────────────────────────────
  → Mesh too fine or solver tolerances too strict
  → Options:
     1. Increase maxCo from 0.7 to 1.0 (less stable)
     2. Coarsen mesh away from fan zone
     3. Increase deltaT to 0.1 s
     4. Run on fewer time steps (endTime = 5 s)

  PROBLEM: Jet velocity unrealistic (<0.5 m/s)
  ─────────────────────────────────────────────
  → Fan pressure coefficient f too low
  → Increase: f from 4.17 to 6.0 or 8.0
  → Re-run from latestTime (startFrom latestTime)

┌─────────────────────────────────────────────────────────────────────────────┐
│ ADJUSTMENT GUIDE                                                            │
└─────────────────────────────────────────────────────────────────────────────┘

  WANT STRONGER FAN?
  ─────────────────
  0/p: Change "f uniform 4.17;" → "f uniform 6.0;"
  (Increases pressure rise from 10 Pa to 15 Pa)

  WANT FASTER SIMULATION?
  ───────────────────────
  controlDict: Change "deltaT 0.05;" → "deltaT 0.1;"
  (Faster but less accurate; may diverge if too large)

  WANT HIGHER ACCURACY?
  ────────────────────
  system/fvSolution: Change "tolerance 1e-6" → "tolerance 1e-7"
  (More iterations, slower but more accurate)

  WANT COARSER MESH?
  ─────────────────
  system/blockMeshDict: Reduce cell counts (e.g., 21 20 35 → 15 15 25)
  Then re-run: blockMesh → topoSet → createPatch → foamRun
